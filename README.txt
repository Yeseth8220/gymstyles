GYM STYLES

Esta pagina web permite el registro e ingreso de usuarios y entrenadores en un gimnasio

Es una pagina monolitica realizada con laravel 8 y que implementa una autenticación por roles (Administrador, Entrenador y Usuario)
Para este proyecto se utilizó codigo html, css, php y javascript

PERMISOS DE CADA ROL

El administrador puede crear, editar y eliminar las clases, entrenadores y usuarios

El entrenador puede crear y visualizar las clases y los usuarios

El usuario solo puede visualizar las clases, registrarse visualizar la pantalla de inicio

Todas estas acciones se ejecutan por medio de un controlador de funciones
